Junior Full Stack Web Developer

Someone who loves coding things, specially websites.

I am a self-taught web developer who has a natural talent and passion for computers. At the moment I leave a double life, working as a Data Analyst during the day and coding my own projects at night. My goal is to launch a full-time web developer career and now I feel more ready than ever. For the last 5 years I have been following a steep learning curve improving my PHP, JavaScript, and CSS skills. At the back-end from a complete spaghetti PHP code to a more OOP structure following the MVC design patterns with the support of frameworks like Laravel. At the front-end from a total chaos of files, to a modular approach using NodeJS and Webpack, and from a clunky redundant coding style to building reusable components using VueJS and React always keeping up with the DRY rule. I am also becoming less dependent to libraries like JQuery and sticking to Vanilla JS whenever it is possible. In addition, due to the nature of my current job I am proficient with relational databases.

Cards

A-maze-ing

A maze generator that uses the backtrack algorithm to find it's way through. It is built in vanilla JavaScript. The link will prompt you to the repository on Github.

SellThatWine

This is the crown jewel. SellThatWine is an online wine market in realtime where buyers and sellers can get in contact and negociate to get the best prices and deals on wine. It is built from scratch by myself using JQuery, PHP, and MySQL, and it has been a great place to progress in my learning curve. In the process I also gained experience with servers, hosts and domains. The next steps include migrating all the existing code to a MVC structure using Laravel, and also implementing online payments.

Glitch

Making my first steps in NodeJS. This little app uses OpenWeatherMap's API (https://openweathermap.org) to randonmly compare the weather across UK towns. This app is hosted on Glitch.

Bubble Sort

This project takes advantage of the flexbox, background position, and the bubble sorting algorithm to create a nice visual effect. Hosted on Codepen.

Temp Gauge

A simple temperature gauge that allows you to check any city in the world thanks to VueJS and OpenWeatherMap's API (https://openweathermap.org). Hosted on Codepen.
